# Premier League Season Simulator

Simulates the remainder of the Premier League season using Elo ratings
and a Monte Carlo approach. Outputs a heatmap of finish probabilities.

## Data

- `Club Elo.csv`: Current Elo ratings per club
- `Matches.csv`: Remaining fixtures
- `PL Table.csv`: Current points table

## Usage

pip install -r requirements.txt
python main.py
